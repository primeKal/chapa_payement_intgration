Chapa Payment Integration

Pay with different options through this payment acquirer

Pay and Verify from Website and Sales

Helps to automate your ecommerce with onlie payment through chapas gatewway

1. Get private and public key from chapa website

2. Install this module

3. Go to settings Payment Acquirers and select chapa. Update the keys and try in testing mode first.

4. Test the ecommerce app and pay with chapa. All good to go.
